> 추후 필요할 때를 대비하여 적어둠

```bash
for file in * ; do iconv -f euc-kr -t utf-8 "$file" > ../ResUtf8/"$file" ; done
```

* 현재 path에 있는 모든 파일 `*` 을 `../ResUtfu/` 에 변환하여 저장
* 특정 파일을 특정 path로 변환하도록 하기위해서는 위 두가지를 변경
